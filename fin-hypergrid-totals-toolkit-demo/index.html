Altering summary data

In dev console, you can alter the summary data with or without the help of the API. To retrieve the first (and only) summary data row, modify the 4th cell, and repaint:

Without the API:

var totals = grid.behavior.subgrids.lookup.topTotals.getRow(0);
totals[3] = 5050;
grid.repaint();

With the API:

var totalRows = grid.behavior.getTopTotals();
var totals = totalRows[0];
totals[3] = 5050;
grid.behavior.setTopTotals(totals);
Note that in this example, the same row data array is being used for both top and bottom, so changing one also changes the other.
